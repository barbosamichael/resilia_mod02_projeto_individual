1 - Objetivo:

	1.0 - Construir um programa que tenha a capacidade de fornercer as informacoes de candidatos com notas
	minimas igual ou superior ao input dado pelo usuario (nota de corte).
	1.1 - Deve retornar os resultados numa lista.
	1.1 - Deve usar uma funcao para consultar os valores.
	1.2 - Subir o arquivo .py no github e fornecer o link onde o arquivo se encontra no portal da Resilia.
	


2 - Glossário:

	2.0 - variaveis fundamentais para o funcionamento do programa e entendimento:
	eX_tX_pX_sX (sendo que cada X eh substituído pela avaliação da pessoa em
	uma das etapas do processo - entrevista, teste teórico, teste prático e
	avaliação de soft skills).

	2.1 - variaveis chaves/fundamentais:
		
		2.2 - e:entrevista

		2.3 - t:teste teórico

		2.4 - p:teste prático

		2.5 - s:avaliação de soft skills

3 - Breve explicacao do fluxo do programa:

	3.0 - funcao main(): eh a responsavel pela estruturacao do programa, chama as funcoes subjacentes e
	ordena os devidos fluxos.
	3.1 - funcao formatacao(): responsavel pela saida padrao do programa (nome do candidato e notas(e,t,p,s)
	3.2 - funcao prints_telas_inicial (): informa as opcoes disponiveis para o usuario captura e escolha inicial
	3.3 - busca_candidatos2(amostra,e,p,t,s): responsável pela consulta e retorno dos resultados.
